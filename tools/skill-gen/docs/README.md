# skill-gen docs

This folder is the handoff package for teams using Codex or Claude Code to build similar converters.

## Included files

- `prd.md`: product requirements and acceptance criteria.
- `skills.md`: reusable generated-skill example/template.
- `codex-claude-usage.md`: copy-paste prompts and workflow for both tools.
- `testing.md`: verification checklist and commands.

## Deployed UI navigation

The deployed Pages artifact includes:

- `index.en.html` (English)
- `index.zh-TW.html` (Traditional Chinese)

## Recommended usage order

1. Read `prd.md` to align on scope and success criteria.
2. Use `skills.md` as the output style reference.
3. Paste instructions from `codex-claude-usage.md` into your AI coding tool.
4. Run checks from `testing.md` before shipping.
