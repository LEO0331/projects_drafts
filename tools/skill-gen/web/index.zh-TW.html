Frontend Skill Forge

把 3 個靜態檔案轉成可重用的 SKILL.md

匯入任意小功能的 index.html、style.css、script.js， 產出可給 Codex、Claude Code 等 AI 工作流使用的結構化技能檔。

等待輸入中。

1. 基本設定

設定輸出名稱與模式。

2. 提供檔案

可上傳檔案、選資料夾，或直接貼內容。

資料夾模式需包含 index.html、style.css、script.js。

3. CLI 指令（選填）

輸入本機資料夾路徑，自動產生 CLI 指令。
